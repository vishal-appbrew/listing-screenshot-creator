# Device Frames

This directory is reserved for custom device frame overlays (PNG files with transparency).

The generator currently draws device frames programmatically using canvas, so no files are required here. You may place custom frame PNGs here for future use.

## Naming Convention

- `iphone-frame.png` — iPhone frame overlay
- `android-frame.png` — Android frame overlay
- `ipad-frame.png` — iPad frame overlay
