# Fonts

Place font files in this directory to enable custom typography in generated screenshots.

## Recommended: Outfit Font

The template engine looks for `Outfit-Bold.ttf` in this directory.

To install:
1. Visit https://fonts.google.com/specimen/Outfit
2. Click "Download family"
3. Extract the ZIP and copy `static/Outfit-Bold.ttf` into this directory

If the font file is not present, the generator falls back to system fonts (Helvetica Neue / Arial).
